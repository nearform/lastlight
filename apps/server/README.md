# Last Light — server package

The Last Light harness + server (npm package `@lastlight/core`). This package
lives at `apps/server/` in the [nearform/lastlight](https://github.com/nearform/lastlight)
monorepo — see the repository root `README.md` for the project overview and
`apps/server/CLAUDE.md` / `spec/` for the development guide and architecture
reference. The published `lastlight` CLI lives separately at `packages/cli/`.

This file exists so the published tarball keeps shipping a README (npm
auto-includes `README.md` from the package root).
